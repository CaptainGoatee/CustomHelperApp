Chat Commands

/me - Used when doing an action - appears over your body. Ex: /me Grabs Wallet
/gme - Same as /me but global. Appears in game chat.
/twt - In-Game Twitter function. Ex: /twt The bank is being robbed at postal 4000!
/fevpn - Same as Twitter but not visible to Law Enforcement. Ex: /fevpn selling drugs at postal 3000
/e - For emotes in our server, Press F3 to get their shortcuts.
/dv - Stand next to any vehicle in our server to delete it. Please roleplay this out.
/help - For a list of these commands any-time in-game.
/calladmin - Used to request staff assistance. Ex: /calladmin I got RDMed by ID 192
/headtag - Used to change the headtag above your player. To blank it out, do /headtag 1
/gangtag - Same as above, for gangs. Blank do: /gangtag 1
/radio - This is a feature we have in game that gives members access to a toggleable radio if you wanted to go into a specific channel and talk privately in-game without having to be in a vc!!
